# 🛒 Multi-Vendor E-Commerce Platform

## Overview

Multi-Vendor E-Commerce Platform is a full-stack web application that allows multiple vendors to add and manage products while enabling customers to browse products, add items to their shopping cart, and place orders. The platform simulates a real-world online marketplace with vendor and customer functionalities.

---

## Features

### Customer Features
- Browse available products
- Add products to cart
- View cart items
- Checkout and place orders
- Real-time product listing

### Vendor Features
- Add new products
- Manage product inventory
- Vendor dashboard

### Admin Features
- View placed orders
- Product management APIs

## API Endpoints

### Products

GET /products
Fetch all products

POST /products
Add new product

### Orders

POST /order
Place order

GET /orders
Fetch all orders
